# 📊 Employee Insights Dashboard

An interactive workforce analytics dashboard built with Python Dash and Plotly to visualize employee trends, monitor productivity, and analyze department-wise performance.

# 🚀 Features

Employees Over Time – Track workforce growth and turnover trends by year and status.

Correlation Analysis – Explore relationships between productivity, work hours, training hours, and monthly income.

Department-wise Analysis – Interactive pie and line charts for average income trends.

Employees & Department Overview – Visualize employee status distribution, department counts, and average tenure.

# 🛠 Tech Stack

Python

Dash (Plotly)

Plotly Express

Pandas

NumPy

# 📂 Dataset

The dataset contains anonymized employee records including:

Name, Age, Gender

Department, Status (Active/Inactive)

Joining Date, Experience

Monthly Income, Training Hours, Work Hours Per Week

Productivity %

# ⚙️ Installation & Usage

1. Clone the repository<br>
  ```git clone https://github.com/your-username/employee-insights-dashboard.git```<br>
  ```cd employee-insights-dashboard```

2. Install dependencies<br>
   ```pip install -r requirements.txt```

3. Run the application<br>
   ```python app.py```

4. Open in browser<br>
   Visit ```http://127.0.0.1:8050``` in your browser.
